# NextAccess

Sistema inteligente de controle de acesso corporativo utilizando QR Code dinâmico, painel web, aplicativo mobile e integração IoT.

## 🚀 Funcionalidades

### Painel Administrativo
- Dashboard
- Cadastro de empresas
- Cadastro de funcionários
- Cadastro de visitantes
- Avisos e notificações
- Logs de acesso

### Aplicativo do Usuário
- QR Code para acesso
- Avisos e notificações

### Aplicativo do Administrador
- QR Code
- Aprovação de visitantes
- Comunicação com recepção
- Avisos

---

## 🔐 Controle de Acesso

O acesso é realizado através de QR Code dinâmico validado em tempo real utilizando webcam conectada ao sistema.

---

## 🛠 Tecnologias

- Next.js
- React
- Node.js
- PostgreSQL
- Socket.IO
- Flutter / React Native
- ESP32 (futuro)

---

## 📌 Objetivo

Desenvolver uma plataforma moderna de gerenciamento de acesso predial corporativo com foco em segurança, automação e integração entre web, mobile e IoT.
